# 第十一章 文件系统的实现——实现细节

## 本讲主题

介绍磁盘空闲空间的四种管理方法。

## 核心知识点

- **位向量法**
  - 用 N 个比特表示磁盘的 N 个块，1 表示空闲，0 表示已占用
  - 通过扫描位向量即可快速找到空闲块
  - 实现简单直观，空间开销固定
- **链接表法**
  - 第一个空闲块记录下一个空闲块地址，以链表形式串联所有空闲块
  - 无额外空间浪费，利用空闲块自身存储指针
  - 缺点：查找连续空闲空间困难，需遍历链表
- **分组法**
  - 第一个空闲块记录后续 N 个空闲块的地址
  - 第 N+1 个空闲块再记录其后 N 个空闲块地址，依此类推
  - 比纯链表法更高效，减少遍历次数
- **计数法（连续分配）**
  - 针对磁盘中连续空闲块的场景
  - 在第一个空闲块中记录后面连续空闲块的个数
  - 适用于空闲块分布较集中的情况

## 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 位向量 | Bit Vector | 用比特数组标记每个块空闲/占用状态 |
| 链接表 | Linked List | 以链表串联所有空闲块的管理方式 |
| 分组 | Grouping | 每组记录 N 个空闲块地址的管理方式 |
| 计数法 | Counting | 记录连续空闲块起始位置和个数的方式 |
| 空闲空间管理 | Free Space Management | 跟踪和管理磁盘可用块的机制 |

## 重要对比/关系

| 管理方法 | 空间开销 | 查找效率 | 连续分配支持 |
|---------|---------|---------|------------|
| 位向量法 | N 比特固定 | 高 | 较好 |
| 链接表法 | 无额外开销 | 低（需遍历） | 困难 |
| 分组法 | 无额外开销 | 中等 | 中等 |
| 计数法 | 低 | 高 | 天然支持 |

## 要点总结

1. **位向量法**用 N 比特标记 N 个块的状态，1 为空闲、0 为占用，查找高效
2. **链接表法**利用空闲块自身存储指针，无额外开销但遍历效率低
3. **分组法**每个块记录 N 个后续空闲块地址，是链表法的**批量优化**
4. **计数法**记录连续空闲块的起始位置和个数，适合空闲块**集中分布**的场景
5. 四种方法各有适用场景，实际文件系统可根据磁盘特性**选择或组合**使用
