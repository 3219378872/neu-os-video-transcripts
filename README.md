## 📖 Introduction (项目简介)
This repository contains the transcripts of online video lectures for the Operating Systems course at the School of Software, Northeastern University (NEU), along with extracted core knowledge points and study notes. 

本项目旨在为东北大学软件学院《操作系统》课程的在线视频提供**完整的字幕转录文本**，并提炼归纳每一章节的**核心知识点**，方便同学们复习、检索和备考。
